Welcome to WAVERSE!!!

by Code of Duty

WAVERSE is an Artificial Intelligence end-to-end pipeline that first identifies potential seismic events and then extracts features for deeper analysis.

By processing NASA data from the Moon and Mars, the project employs the following models to classify seismic events.

  • XGBoost

  • Recurrent Neural Network (RNN)

Note: these were trained by using Google Colab and in the case of RNN, the GPU option.

The final output is a catalog of true seismic events along with detailed plots for analysis which are deployed in an user-friendly interface.

Improvements and Extensions

The main setback regarding the analysis had to do with the scarcity of data, especially in Mars. Therefore, for the first improvement, we would find more information about these phenomena.

Secondly, for a more nurtured visual experience, we would include more dynamic dashboards that are aligned in real-time with the proper EDA and analytical processes.

Finally, we wish to expand these analysis to the Earth, Sun and in general, any planet whose information is available.
